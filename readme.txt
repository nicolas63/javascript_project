1. Installer les dépendances des deux dossiers

2. Lancer le serveur à partir de Node_TP2/ avec la commande : node --harmony --use_strict server/index.js

3. Créer le fichier bundle.js à partir de Pratice4-React/ avec la commande : node_modules/.bin/webpack -w --config
