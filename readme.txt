1. Installer les dépendances de node avec npm install dans les dossiers server/test et front

2. Lancer le serveur avec la commande node --harmony --use_strict server/index.js

3. Générer le fichier bundle.js depuis le dossier front avec la commande node_modules/.bin/webpack -w --config webpack.config.js

4. Ouvrir front/index.html avec votre navigateur favori

Pour lancer les tests : ./run_test.sh depuis le dossier server
